# Deployment / Remote Access Notes

This project is now intended to live in its own standalone Git repository.

## Goal
Access and manage this project independently from your laptop and keep it separate from the larger workspace repository.

## Recommended remote path
- create a dedicated GitHub repo for `website-factory`
- push this folder as its own repository
- use GitHub as the source of truth for laptop access

## Notes
This project is currently documentation/workflow heavy and may later become its own app or internal tool.
