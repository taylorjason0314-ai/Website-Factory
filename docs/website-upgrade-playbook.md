# Website Upgrade Playbook

## Purpose
This is the business and prospecting playbook for Website Factory.

Use it for:
- identifying weak websites
- understanding what to look for
- structuring outreach angles
- thinking through the offer and prospect fit

It is not the source of truth for Operator MVP build semantics.

## Core principle
Do not sell generic websites.
Sell:
- more leads
- better mobile conversion
- clearer trust
- easier quote flow
