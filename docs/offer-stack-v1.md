# Website Factory Offer Stack v1

Status: Locked for initial outbound and fulfillment testing

## Positioning
We help local service businesses turn weak websites into lead-generating websites.

This is not positioned as:
- custom design for design's sake
- generic web development
- vague AI website magic

This is positioned as:
- more quote requests
- better mobile conversion
- stronger homeowner trust
- clearer service-area positioning
- a site that helps book jobs

## Initial Niche
- city: San Jose
- niche: landscaping

## Offer 1 — Quick Conversion Refresh
Best for businesses with an existing site that is clearly underperforming.

### Deliverables
- mobile-first homepage refresh
- CTA upgrade (call / quote / estimate)
- improved header + hero messaging
- trust block added or improved
- service-area clarity
- contact/quote flow cleanup
- speed + layout cleanup where practical

### Promise
Improve conversion and clarity fast without forcing a full rebuild.

### Sales Use
Use this as the easiest first yes.

## Offer 2 — Lead Gen Website Rebuild
Best for weak or outdated sites that need a more complete fix.

### Deliverables
- full conversion-focused website rebuild
- homepage + core service pages
- quote funnel/contact flow
- trust and social proof sections
- before/after gallery structure
- service-area messaging
- mobile-first layout

### Promise
Turn the website into a real sales asset instead of a digital brochure.

### Sales Use
Use when the current site is weak enough that patching it is the wrong answer.

## Offer 3 — Ongoing Growth Retainer
Best for clients after initial project close.

### Deliverables
- landing page additions
- ongoing conversion improvements
- local SEO page additions
- seasonal updates/offers
- analytics review and changes

### Promise
Keep improving lead generation instead of letting the site go stale again.

## Offer Selection Rule
- If the site is salvageable: start with Quick Conversion Refresh.
- If the site is badly outdated or structurally weak: pitch Lead Gen Website Rebuild.
- If client closes and wants continued help: upsell Ongoing Growth Retainer.

## What We Are Really Selling
- more inbound leads
- more calls and quote requests
- better mobile conversion
- more trust from homeowners
- clearer reasons to choose the business

## Do Not Drift Into
- cheap commodity website talk
- broad branding consulting
- custom software promises
- technical complexity in the pitch
