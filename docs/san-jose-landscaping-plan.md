# San Jose Landscaping Plan

## Goal
Build the first niche-specific outbound system for San Jose landscaping businesses with weak websites.

## Immediate Steps
1. collect 25–50 landscaping businesses in San Jose
2. score each website using the lead-gen rubric
3. identify the top 10–20 weakest/highest-opportunity prospects
4. prepare a short outreach sequence
5. test messaging

## What Makes a Great Landscaping Prospect
- older-looking site
- weak before/after showcase
- no obvious quote funnel
- poor mobile layout
- weak service-area messaging
- no strong customer proof

## Desired Outcome
Close the first few website upgrade clients and turn the process into a repeatable lead machine.
